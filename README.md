# Pandas & Machine Learning Learning Project

## Overview

This project is a comprehensive learning repository for data analysis and machine learning using Python. It covers fundamental Pandas operations, data visualization, exploratory data analysis (EDA), and linear regression implementation using scikit-learn. The project includes multiple Jupyter notebooks demonstrating progressive concepts from basic DataFrame operations to complete machine learning workflows.

## Technologies Used

- **Python 3**
- **Pandas** - Data manipulation and analysis
- **Matplotlib** - Data visualization
- **Seaborn** - Statistical data visualization
- **scikit-learn** - Machine learning library for linear regression

## Project Structure

### Notebooks

1. **[1.ipynb](1.ipynb)** - Pandas Fundamentals

   - Basic Pandas operations and DataFrame creation
   - Creating DataFrames from lists, dictionaries, and nested structures
   - Reading CSV files
   - Data filtering and selection
   - Correlation analysis with heatmaps
   - Working with Screen Time Data for practical examples

2. **[2.ipynb](2.ipynb)** - Pandas Profiling

   - Introduction to ydata_profiling
   - Loading and exploring credit card dataset from external source

3. **[3.ipynb](3.ipynb)** - Data Loading Practice

   - Working with Churn Modelling Dataset
   - Basic data exploration and shape inspection

4. **[4 scikit-learn.ipynb](4%20scikit-learn.ipynb)** - Machine Learning with scikit-learn

   - Linear regression implementation
   - Working with NASDAQ-100 stock data
   - Model training and prediction
   - Model evaluation metrics (MSE, MAE, R² score)
   - Data visualization with matplotlib
   - Salary prediction using linear regression

5. **[Assignment 01 - Salary Prediction.ipynb](Assignment%2001%20-%20Salary%20Prediction.ipynb)** - Complete ML Project
   - End-to-end salary prediction project
   - Exploratory Data Analysis (EDA)
   - Feature engineering
   - Linear regression model building
   - Model evaluation and interpretation
   - Loss calculation and visualization

### Datasets

1. **[Screen Time Data.csv](Screen%20Time%20Data.csv)** - Personal screen time statistics containing:

   - Date and week day information
   - Total screen time
   - Activity breakdown: Social Networking, Productivity, Reading and Reference, Entertainment, Health and Fitness, Creativity, Yoga, and Other
